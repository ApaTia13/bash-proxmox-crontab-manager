# bash-proxmox-crontab-manager
Интерактивный Bash-менеджер для управления задачами cron в Proxmox. Скрипт позволяет управлять VM и LXC через cron: добавлять, редактировать и удалять задачи, проверять их состояние, логировать выполнение и отправлять уведомления об ошибках в Telegram. Работает с пользовательским crontab и использует обёртку для безопасного выполнения команд.
